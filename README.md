# 👥 KeenKeeper — Keep Your Friendships Alive


## 📖 Project Description
KeenKeeper is a modern and fully responsive web application built with Next.js that helps users manage and maintain their friendships effectively. Users can track their interactions (calls, texts, video chats), view detailed friend profiles, and monitor communication habits through a timeline and analytics dashboard.

The project focuses on performance, scalability, and a smooth user experience using Next.js features like routing, loading UI, and component-based architecture.


## 🛠️ Technologies Used
⚡ Next.js (App Router)  
🎨 Tailwind CSS  
🔣 React Icons  
📊 Recharts  
🔔 React Toastify  
🚀 Netlify  
🧠 JavaScript (ES6+)  


## ✨ Key Features
🔹 Dynamic Routing with Next.js — Navigate to individual friend detail pages using dynamic routes  
🔹 Interaction Tracking System — Log Call, Text, and Video interactions with toast notifications and timeline updates     
🔹 Responsive & Optimized UI — Fully responsive design with loading states, 404 page, and smooth user experience
